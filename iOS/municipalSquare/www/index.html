Week 1

1. Less obvious perks of using 'live' data resources include logins and payments. How could Twitter be used in these two scenarios?

Using Twitter as a login portal can make for a better user experience, taking less time and getting the user to using your app quickly. Using the API to make payments it speeds up the process of purchasing an item you want, and we all know that convenience can increase sales.

2. Visit Programmableweb.com and search for JSON-P data sources. Why would we need JSON-P, specifically?

JSON-P is JSON with padding. I gives us a method of requesting data from different domains helping us around the "same origin policy". This ability to retrieve data from different sites makes a richer more dynamic applications.

3. Based on the JSON-P search results: what data resource(s) do you think might pair well with which types of Native device features?

I am very interesting in using feed from Twitter and Facebook, and pairing it with the native device features of geolocation and maps to give a user news and updates on what is going on around them. Using notification in combination with the WeatherBug for alerts is another combination I have thought about.

4.Please describe any design and functionality intentions you may have for your Demo app components this month.

Currently, I am working on a more making the layout of my app more responsive, so that the way the app look depends on how big the screen of the the device. In the latest version of jQuery Mobile, there is a demo for a grid-listview that I am trying to implement. I'm trying to keep my app as simple as possible and expand its features as I learn more.

5. Briefly describe some past 'lessons learned' regarding workflow as it relates to research, design, development, etc.

Formatting my code in a way that I can easily sift through no matter how long ago you wrote it is a must. I try to leave plenty of of informative comments on sections of code I write to help me along and to remind me of what I was trying to do. I need to work on my research methods and putting information that I find online together better to help me learn quicker and get things done. I also need to work on networking with friends and people that can help when I am stuck. Asking for help is tough for me.
